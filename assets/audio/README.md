# 音效文件說明

由於技術限制，這裡是音效文件的說明。
在實際部署時，請添加以下音效文件到 assets/audio/ 目錄：

1. magic-bg.mp3 - 背景音樂，輕柔的魔法音樂
2. portal.mp3 - 門戶轉場音效
3. click.mp3 - 點擊音效
4. calendar.mp3 - 行事曆切換音效
5. birthday-discover.mp3 - 發現生日驚喜音效

## 獲取免費音效資源：

1. **Freesound.org** - 免費音效庫
2. **Zapsplat.com** - 高品質音效
3. **YouTube Audio Library** - 免費音樂和音效
4. **Bensound.com** - 免費背景音樂

## 推薦音效類型：

- 魔法背景音樂：輕柔的鋼琴或弦樂
- 門戶音效：傳送門或魔法的"woosh"聲
- 點擊音效：清脆的鈴聲或魔法火花聲
- 行事曆音效：頁面翻轉聲
- 生日發現音效：歡樂的驚喜聲或煙火聲

音效文件應為 MP3 格式，大小控制在 1MB 以內以確保載入速度。
