# 魔法生日行事曆

一個精緻的生日慶祝網頁，一開始看起來像普通行事曆，但點擊生日事件會打開哈利波特主題的慶生頁面。

## 功能特色

- 📅 互動式行事曆界面
- 🎂 點擊生日日期打開慶生模態框
- 🧙‍♂️ 哈利波特主題設計
- ⏰ 時間轉換器（麻瓜時間 ↔ 巫師時間）
- 🚂 9¾ 月台元素
- 🎉 彩紙動畫效果

## 如何使用

1. 開啟 `index.html` 在瀏覽器中。
2. 瀏覽行事曆，找到標記為生日的日期（預設為每月15日）。
3. 點擊生日日期，打開慶生頁面。
4. 享受魔法般的生日驚喜！

## 自訂設定

- 在 `script.js` 中修改 `birthdayDay` 變數來設定生日日期。
- 替換 `index.html` 中的圖片 URL 為真實的哈利波特相關圖片。
- 在 `styles.css` 中調整顏色和樣式以符合您的喜好。

## 技術棧

- HTML5
- CSS3
- JavaScript (ES6+)
